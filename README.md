# Sistema de Control de Stock

> Aplicación de escritorio robusta para la gestión integral de inventario, ventas y facturación.

Este proyecto fue desarrollado con el objetivo de crear una solución eficiente para pequeños comercios, permitiendo administrar productos, registrar ventas y generar reportes exportables. El foco principal del desarrollo fue la implementación de **Buenas Prácticas de Programación** y principios sólidos de **POO**.

---

##  Tecnologías y Conceptos Aplicados

El sistema no solo gestiona datos, sino que demuestra la implementación práctica de conceptos avanzados de desarrollo:

* **Lenguaje:** C# (.NET)
* **Interfaz:** Windows Forms
* **Programación Orientada a Objetos (POO):** Uso extensivo de **Herencia** y **Polimorfismo** para una arquitectura escalable.
* **Persistencia de Datos:** Manejo de archivos **JSON** para almacenamiento local y recuperación de datos (Serialización/Deserialización).
* **Reportes:** Generación automática de archivos **CSV** para análisis de datos externos (Excel).

##  Funcionalidades Principales

### 1. Panel de Control y Visualización
El sistema ofrece una vista clara del estado actual del stock. Permite visualizar el detalle específico de cada producto ingresado en el sistema.

### 2. Gestión de Productos (CRUD)
Administración completa del ciclo de vida de los productos.

* **Alta de Productos:** Formulario intuitivo para ingresar Nombre, Cantidad (Kg) y Descripción.
![Alta de Producto](https://github.com/user-attachments/assets/5959a7aa-758b-45c5-909c-bc4873a1da28)

* **Modificación:** Capacidad de editar las propiedades de un producto existente.
![Modificar Producto](https://github.com/user-attachments/assets/ba93fca0-0f62-4060-913d-48e4161c3ec3)

* **Baja de Productos:** Eliminación segura de productos del inventario.
![Eliminar Producto](https://github.com/user-attachments/assets/8b2911c7-16e4-4f18-8822-bbd3a804add2)

---

### 3. Registro de Ventas
Módulo dedicado a la salida de mercadería. El usuario selecciona el producto, ingresa la cantidad vendida (Kg) y el monto de la transacción, actualizando automáticamente el stock disponible.

![Registro de Venta](https://github.com/user-attachments/assets/e879a400-6af4-411a-9bfd-a71d123d603b)
